About

This project involved an analysis of weather patterns in relation to proximity to the equator. A random selection of 565 cities was drawn from citipy and the weather data from these cities was obtained using an API key and the site Open Weather. The variables analysed were temperature, humidity, cloudiness, and windspeed. The visualisations presented here were created with Matplotlib.
